2.0 将项目提交到 git 中
	2.1 将项目进行 git 初始化: git init
	2.2 查看当前项目状态： git status
	2.3 将没有添加的文件进行 add: git add .
	2.4 将文件提交到本地仓库（版本库）： git commit -m '完成了首页&分类&搜索列表'
	2.5 建立远程仓库：
		github git@github.com:hubgitjiang/pingyougou29.git
		码云：
	2.6 将运程仓库与本地进行链接： git remote add origin git@github.com:hubgitjiang/pingyougou29.git
	2.7 与运程的 master 建立联系： git push -u origin master

	小程序也有生命周期：
		微信：
			小程序：
				onLauch: 当小程序加载时执行
				onShow: 当小程序显示时执行
			页面：
				onLoad: 页面加载时执行
				onReady: 当页面渲染时执行
				onShow: 当页面显示时执行
				onHide: 当页面隐藏时执行
				ohUnload: 当页面卸载时执行
